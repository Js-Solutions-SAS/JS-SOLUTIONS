# Glossary

- mission: impacto primario del agente.
- inputs: informacion minima requerida para ejecutar.
- outputs: entregables esperados del agente.
- workflow: secuencia de trabajo recomendada.
- quality_gates: validaciones antes de cerrar.
- handoff: informacion para el siguiente rol.
- escalation: condiciones para levantar bloqueo.
