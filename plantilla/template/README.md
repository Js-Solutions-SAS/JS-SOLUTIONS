# Landing generada

Sitio Astro estatico generado desde `/plantilla`.

## Comandos

```bash
npm install
npm run build
npm run check:seo
npm run dev
```

## Deploy en Vercel

- Root directory: esta carpeta.
- Build command: `npm run build`.
- Output directory: `dist`.
- Configura el dominio real del cliente y actualiza `src/data/client.config.ts` si cambia el dominio.
