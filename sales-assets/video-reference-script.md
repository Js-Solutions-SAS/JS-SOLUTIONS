# Video Reference Script

## Video corto: 90 segundos

**Objetivo:** vender la idea de sistema comercial operativo para PyMES, usando pantallas reales del repo como prueba.

**0-10s - Problema**

"Muchas PyMES no necesitan solamente una pagina web. Necesitan que cada lead que llega por Google, WhatsApp o formulario tenga seguimiento, cotizacion y una siguiente accion clara."

**10-25s - Landing**

"En JS Solutions estamos armando este sistema desde la primera pantalla: una landing enfocada en conversion, con rutas por tipo de cliente, CTA directo a WhatsApp y un cotizador que ayuda a entender alcance antes de hablar con ventas."

**25-40s - WhatsApp y cotizacion**

"El flujo no termina en un mensaje. El lead se califica, se captura la fuente, se puede generar una cotizacion y se deja todo listo para seguimiento comercial sin depender de una hoja manual."

**40-58s - Admin cockpit**

"Por dentro, el equipo no trabaja a ciegas. El admin concentra cotizaciones, entregas, capacidad, aprobaciones, cambios, SLA, finanzas, portafolio, riesgos y SOPs. Eso convierte ventas y entrega en una operacion trazable."

**58-73s - Portal cliente**

"Del lado del cliente, el portal permite ver estado, completar brief, aprobar documentos o entregables y seguir el proyecto sin pedir actualizaciones por chat todo el tiempo."

**73-85s - Arquitectura**

"La diferencia es que n8n automatiza alrededor de un API de negocio. El API guarda leads, cotizaciones, contratos, proyectos, pagos y eventos. Eso hace que la automatizacion sea auditable."

**85-90s - CTA**

"Si tu negocio recibe leads pero pierde seguimiento, podemos replicar este sistema para tu operacion. Escribenos y revisamos tu flujo actual."

## Video extendido: 3 minutos

**0-15s - Hook**

"Una pagina web sola no arregla ventas. Si despues del formulario nadie responde a tiempo, si las cotizaciones se pierden o si el cliente no sabe en que va su proyecto, la web solo esta decorando el problema."

**15-35s - Resultado deseado**

"El objetivo es crear una operacion comercial digital: atraer, capturar, calificar, cotizar, aprobar, cobrar y entregar con trazabilidad. Eso es lo que estamos construyendo en JS Solutions como sistema replicable para PyMES."

**35-55s - Landing y demanda**

"La landing funciona como motor de demanda. Tiene mensajes por segmento, una ruta para PyMES, contenido tecnico suficiente para dar confianza y CTA hacia WhatsApp o cotizador. La promesa es simple: que el prospecto entienda rapido que problema resolvemos y pueda dar el siguiente paso."

**55-75s - Cotizador e intake**

"El cotizador convierte una conversacion vaga en informacion accionable. En vez de preguntar todo por chat, el sistema guia el alcance, genera una prelectura comercial y permite crear una oportunidad con contexto."

**75-105s - API y automatizacion**

"La parte importante esta debajo. El API no es un puente improvisado a n8n; es el core del negocio. Persiste leads, briefs, cotizaciones, contratos, pagos, proyectos, aprobaciones, tickets y eventos. n8n entra como orquestador: genera documentos, dispara seguimientos, conecta pagos o firma, pero no reemplaza la fuente de verdad."

**105-130s - Admin y operacion**

"El admin es el cockpit interno. Desde ahi el equipo puede operar cotizaciones, entregas, aprobaciones, cambios, SLA, capacidad, portafolio, finanzas, riesgos y SOPs. Eso permite vender mas sin que la operacion se vuelva fragil."

**130-150s - Portal cliente**

"El portal cierra el circuito: el cliente ve estado, carga informacion, aprueba entregables y entiende que sigue. Eso baja friccion, reduce mensajes repetidos y da una experiencia mas profesional."

**150-165s - Prueba tecnica con Graphify**

"Para revisar y explicar el sistema estamos usando Graphify, que convierte el repo en un grafo consultable. Nos ayuda a ver como se conectan landing, portal, admin, API y automatizaciones, y a convertir arquitectura tecnica en contenido comercial claro."

**165-180s - CTA**

"Si tienes una PyME con leads por WhatsApp, formularios o redes, pero el seguimiento depende de memoria, chats y hojas sueltas, este sistema se puede adaptar a tu operacion. Primero auditamos tu flujo, luego montamos una version inicial y dejamos trazabilidad desde el primer contacto hasta la entrega."

## Promesas permitidas

- "Reducir friccion comercial."
- "Mejorar trazabilidad."
- "Evitar que leads queden sin siguiente accion."
- "Centralizar cotizacion, aprobacion y seguimiento."
- "Dar una experiencia mas profesional al cliente."

## Promesas que no deben usarse sin evidencia

- "Duplicamos tus ventas."
- "Garantizamos X clientes nuevos."
- "Recuperamos la inversion en X dias."
- "Automatizamos todo sin intervencion humana."
